


## Exercice 1 : Peut-on utiliser le polymorphisme : 
Oui, on peut regrouper le code dans des classes pour mieux organiser le programme.

Le polymorphisme est possible si plusieurs classes utilisent les mêmes méthodes avec des comportements différents, par exemple une classe client et une classe serveur avec des méthodes comme envoyer ou recevoir.

### Application de chat : UDP ou TCP 
TCP est mieux adapté pour une application de chat car :
Les messages doivent arriver dans l'ordre
On ne peut pas perdre des messages
TCP garantit la fiabilité de la transmission
UDP ne garantit ni l'ordre ni la livraison des messages, ce qui est inacceptable pour un chat
